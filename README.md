# YouTube Watch Time Bot

This bot runs in a Docker container on Hugging Face Spaces.

## Setup

1. Create a new **Space** on Hugging Face.
2. Select **Docker** as the SDK.
3. Upload the files from this repository.
4. The bot will start automatically.

**Note**: To keep the space from sleeping, you may need to add a simple web server (e.g., Express) that listens on port 7860.
